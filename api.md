Полная документация доступна автоматически через встроенный интерфейс FastAPI:
- **Swagger UI**: http://127.0.0.1:8000/docs

Там вы найдёте:
- Описание всех эндпоинтов
- Форматы запросов/ответов
- Возможность тестирования прямо в браузере


### Предварительные требования
# Установка зависимостей (выполнить один раз)
pip install -r requirements.txt

### Шаг 1: Запуск прогнозного API

После запуска проверьте:
→ Откройте http://127.0.0.1:8000/health — должен вернуться статус {"status": "ok"}
→ Откройте http://127.0.0.1:8000/docs — полная документация эндпоинтов

### Шаг 2: Запуск веб-приложения (Модуль Г.2)

python app.py

После запуска проверьте:
→ Откройте http://127.0.0.1:5000 — откроется интерфейс планирования маршрутов
